# 数据类型

Python内建的好几种数据类型，而变量则可以储存不同数据类型且可以实现不同的东西，内建的数据类型包括

字符串(文字) 类型:      ```str```<br>
数字 类型：             ```int, float, complex``` <br>
序列 类型：             ```list, tuple, range``` <br>
映射 类型：             ```dict``` <br>
集合 类型：             ```set, frozenset``` <br>
布尔 类型：             ```bool``` <br>
二进制 类型：           ```bytes, byterray, memoryview``` <br>
无 类型：               ```NoneType``` <br>

?> 如何得到数据的类型
```python
x = "alfred"
print(type(x))
# type方法将返回数据的类型
```
